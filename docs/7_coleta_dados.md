## 1. Identificação de Necessidades dos Usuários e Requisitos de IHC:
### Dados sobre o usuário:
- Demografia: Renda, Meio de deslocamento, Idade, Localização
- Experiencia com produtos semelhantes: Frequencia de uso de apps de viagens, experiencias positivas e negativas
- Tecnologia Disponível: Qual modelo de smartphone utiliza
- Outros: colocar em escala o quão receptível o usuário entrevistado é a dividir viagens para a FEI com outros alunos.

 ### Dados de tarefa:
- Objetivos: Quais os objetivos de cada persona usando esse aplicativo?
- Frequência: Nossa proposta justifica um uso frequente do aplicativo?
- Dores: Quais as dificuldades atuais que justificam o uso do aplicativo?
- Gravidade dos erros: qual o impacto nos usuários caso ocorra um mau funcionamento do sistema?  

### Motivações e Valores:
- Atitudes e valores: 
  - o usuário consegue reconhecer os beneficios da nossa proposta? 
  - O Usuário está disposto a viajar com outros passageiros?
  - Qual a distancia maxima que um motorista estaria disposto a viajar para oferecer uma carona?
  - Estaria disposto a sacrificar conforto em prol de economia monetária?
